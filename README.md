# Ecommerce-Sales-Dashboard
Power BI dashboard for analyzing e-commerce sales with insights on profit, sales, customers, and trends.
🛒 E-Commerce Sales Dashboard (Power BI)

📊 Overview

This project is an interactive Power BI dashboard for analyzing e-commerce sales performance.

🚀 Features

- Total Sales, Profit, Quantity & AOV
- State-wise sales analysis
- Category & Sub-category insights
- Monthly profit trends
- Payment mode distribution

🧰 Tools Used

- Power BI
- Excel / CSV Dataset

📁 Project Files

- "ecommerce_dashboard.pbix" → Power BI file
- "dataset/" → Data files
- "dashboard.png" → Dashboard preview

📸 Dashboard Preview

"Dashboard" (dashboard.png)

▶️ How to Use

1. Download the ".pbix" file
2. Open in Power BI Desktop
3. Refresh data if needed
